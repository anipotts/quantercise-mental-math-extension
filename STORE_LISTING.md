## Short Description (132 characters max)

## Detailed Description
Practice mental arithmetic with Quantercise Mental Math — a simple, focused extension for sharpening your calculation skills.

**Quick Drill Mode**
• 2-minute timed sessions with unlimited arithmetic problems
• Addition, subtraction, multiplication, and division
• Instant feedback on every answer
• Track your score, accuracy, and questions per minute

**Simple & Distraction-Free**
• Clean, minimal interface
• Light and dark theme support
• Optional sound effects
• Works completely offline

**Track Your Progress**
• Personal best scores saved locally
• Session history (last 20 drills)
• Questions per minute and accuracy stats

**Privacy First**
• All data stored locally on your device
• No accounts required
• No tracking or analytics
• No internet connection needed

Perfect for students, professionals, or anyone wanting to keep their mental math sharp. Just click the extension icon and start practicing.

Built by Quantercise — https://quantercise.com

## Privacy Tab Answers

### Single Purpose Description
Mental math practice tool with timed arithmetic drills and local score tracking.

### Permission Justification: storage
Required to save user preferences (theme, sound settings) and scores (best score, session history) locally on the user's device. No data is transmitted externally.

### Host Permission Justification
This extension does not request any host permissions.

### Remote Code
This extension does not use any remotely hosted code. All JavaScript is bundled within the extension package.

### Data Usage
This extension does not collect, transmit, or share any user data. All data (preferences, scores) is stored locally using Chrome's storage API and never leaves the user's device.
